# Orb Examples

This repository contains examples of Jade Platform applications that utilise the Orb web framework.

## Index of Examples

* **[JV Summit App](JVSummitApp)** - Event management application that was used at the Jade Velocity 2025 Summit. It features a live-updating agenda with rating system, and live chat.

## Documentation

Each subdirectory of this repository contains a readme file with instructions on how to use the respective example application.
