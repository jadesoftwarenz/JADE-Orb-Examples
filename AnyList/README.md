# AnyList

## Prerequisites

- Jade Platform 2025

## Setup

### Orb

1. Download the Orb framework schema.
2. Select `File` -> `Schema Load` on the menu bar to import the schema.
3. Import Orb's `.scm` and `.ddx` files.

### Project

1. Clone the repository.
2. Select `File` -> `Schema Load` on the menu bar to import the schema.
3. Import project's `.scm` and `.ddx` files.
4. Browse classes of `AnyListSchema` in Jade Platform.
5. Expand to `Object` -> `Application` -> `RootSchemaApp` -> `AnyListSchema`.
6. Open the `init` method.
7. Ensure the `Root` constant points to the path where the templates and other assets are in.
8. Select `File` -> `Schema Load` on the menu bar to start the application.
9. Select the `AnyList` application.
10. Open http://localhost:8000 in your browser.
